An incorrect session-length setting can have a negative impact on session-based analysis; for example, if your users would normally have more than 30 minutes of inactivity during a session, a shorter timeout would divide single-session data into multiple sessions and provide an inaccurate picture of activity.

Set campaign timeout to correspond to the length of time you want to be able to attribute a session or conversion to one of your campaigns. For example, if you run seasonal campaigns, set this value to 3 Months; if you run weekly campaigns, set this value to 7 Days.
